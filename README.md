# SPA-vuex

> A Vue.js project

## Build Setup

### Install and use REST server (at localhost:3003)
simple-rest-server =>
````
npm install
npm start
````
select "f"

### Install and use Socket server (at localhost:3000)
socketio-chat =>
````
npm install
node index.js
````

### Install dependencies and serve client side
``` 
npm install
npm run dev
```
